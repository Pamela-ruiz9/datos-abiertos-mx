# ¿Cómo contribuir?

¡Gracias por querer contribuir! Este diccionario crece con la comunidad.

## Agregar una nueva fuente

1. **Forkea** este repositorio
2. Crea un archivo en `catalogo/<sector>/nombre-fuente.md`
3. Usa la plantilla de [`docs/schema.md`](docs/schema.md)
4. Abre un **Pull Request** con el título: `[nueva fuente] Nombre de la fuente`

## Mejorar una ficha existente

- ¿Encontraste un link roto? Ábrelo como **Issue** con la etiqueta `link-roto`
- ¿La información está desactualizada? PR directo con los cambios
- ¿Falta un snippet de código? ¡Bienvenido!

## Estándares de calidad

- Verificar que los links funcionen antes de hacer PR
- Incluir al menos: URL, formato, periodicidad y un ejemplo mínimo de uso
- Anotar limitaciones o advertencias conocidas — eso es tan valioso como el acceso
- No incluir credenciales, tokens o datos privados
